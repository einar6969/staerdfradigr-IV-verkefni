## 💡 Philosophy

> Most websites are built like **projects**.  
> I build them like **systems**.

### System qualities
- **Understandable** — clear structure, no hidden complexity
- **Extensible** — easy to evolve over time
- **Maintainable** — minimal technical debt

### Principles
- **Structure before visuals** — architecture first, aesthetics second
- **SEO is infrastructure** — not a feature, but baked in from day one
- **Simple over clever** — prefer boring, reliable systems
- **Build for years** — not just launch day
